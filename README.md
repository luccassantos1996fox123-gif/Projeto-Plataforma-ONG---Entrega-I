 Entrega I — Plataforma ONG

Projeto da disciplina **Fundamentos da Web e Estruturação de Interfaces**.  
Objetivo: criar a estrutura base de uma plataforma para uma ONG, usando **HTML5**, **CSS3** e **JavaScript**.

---

   Páginas
- **index.html** — página inicial com informações da ONG  
- **projetos.html** — mostra os projetos e campanhas  
- **cadastro.html** — formulário de cadastro de voluntários e doadores  

---

   Recursos
- Estrutura **HTML5 semântica**  
- Formulário com validação nativa e máscaras (CPF, telefone, CEP)  
- Layout **responsivo** e leve  
- Imagens otimizadas e carregamento `lazy`  
- Ano automático no rodapé  

---

## Como visualizar
1. Abra a pasta do projeto no **VS Code**.  
2. Use a extensão **Live Server** para ver no navegador.  
3. Valide os arquivos no **[W3C Validator](https://validator.w3.org/)**.
